API 文档

Base URL
POST /api/generate

创建邮箱，例如 {"custom":"test","retention_hours":24}

GET /api/mailboxes

获取所有邮箱和邮件数量。

GET /api/mailbox/:address

查看指定邮箱的全部邮件。

GET /api/mailbox/:address/wait?timeout=30

等待指定邮箱的新邮件。

DELETE /api/mailboxes/:address

删除邮箱和该邮箱下的全部邮件。

如果服务器启用了 API_KEY，请在请求头里带上 x-api-key， 或使用 Authorization: Bearer <key>。 页面右上角的 “API Key” 按钮会把 key 存到浏览器本地，之后页面请求会自动带上它。
