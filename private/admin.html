API 文档

Base URL
POST /api/generate

创建邮箱，例如 {"custom":"test","domain":"dart.lat","retention_hours":24}

GET /api/mailboxes

获取全部邮箱和邮件数量。

GET /api/mailbox/:address

查看指定邮箱的全部邮件。

GET /api/mailbox/:address/wait?timeout=30

等待指定邮箱的新邮件。

DELETE /api/mailboxes/:address

删除邮箱和该邮箱下的全部邮件。
